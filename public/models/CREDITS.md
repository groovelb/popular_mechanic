## Download Instructions

1. Visit the Sketchfab model page linked above
2. Create a free Sketchfab account if you don't have one
3. Click "Download 3D Model"
4. Select "glTF" format
5. Extract the ZIP file
6. Rename the `.glb` file to `plymouth-fury.glb`
7. Place it in this folder (`/public/models/`)
